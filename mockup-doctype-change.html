Launchers on Linux

03.09.09

Going from Windows + Launchy to Linux + ??? has always bugged me.

No launcher I’ve found is as even as close to as good as Launchy on Windows. It does three things really well:

  • learns what I want and recalls it when I start typing
  • navigates folders intuitively so I don’t have to launch an Explorer window and then navigate to the folder
  • type in the executable and then tab to send it a file to open on, as in “firefox gmail.com” (actually, I tend to use this most w/ Internet Explorer and just paste in the URL of whatever page I need to check)
